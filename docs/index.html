What is Glider.js?

Glider.js was born out of a frustration for carousels on mobile devices. Inspired by Slick.js, Glider.js aims to be a mobile-focused, fast, lightweight, responsive, dependency-free "carousel". The term carousel is used loosely, as Glider.js lacks some basic carousel features, primarily "infinite" looping and autoplay. Glider.js does not aim to replace the de-facto homepage image carousel, but to enhance the list browsing experience on mobile while retaining functionality with desktop browsers.

Features

  • Insanely Small (1.7kb gzipped)
  • Ridiculously Fast (~15ms initilization)
  • Vanilla JS - No Dependencies (look Mom, no jQuery!)
  • Native Browser Scrolling (momentum scrolling on mobile)
  • Fully Responsive (breakpoint based settings)
  • Customizable Arrow and Dot Navigation
  • Flexbox Support (enabled by default)
  • Easily Extendable
  • Custom Events
